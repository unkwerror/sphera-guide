# Задачи по GIF для сайта

- **Hero / быстрый обзор:** открыть Сферу в браузере -> выбрать пространство -> перейти в `Мои задачи` -> остановиться на списке активных карточек.
- **Что делать в первый день:** открыть `Мои задачи` -> переключить `Активные` / `Планируемые` -> открыть одну карточку -> вернуться в список.
- **Как делать задачи / создание карточки:** нажать `Создать задачу` -> заполнить `Название`, `Описание`, `Приоритет`, `Исполнитель`, `Оценка`, `Спринт` -> сохранить карточку.
- **Экраны, которые тебе нужны / навигация:** последовательно перейти `Мои задачи -> Бэклог -> Канбан -> Структура задач -> Учёт времени` без пауз на мелких кликах.
- **Как рекомендовано строить спринт:** открыть `Бэклог` -> создать спринт -> заполнить даты и название -> перенести 2-3 задачи в новый спринт.
- **Канбан-доска и стендапы:** открыть канбан -> перетащить карточку из `Открыта` в `В работе` -> затем в `Решена` -> показать, что статус обновился.
- **Структура задач и доска:** открыть `Структура задач` -> создать новую структуру -> добавить уровни (эпик/задача/подзадача) -> сохранить.
- **Учёт времени и фильтры:** открыть карточку задачи -> списать время -> перейти в `Учёт времени` -> применить фильтр по исполнителю или статусу.

## Техтребования к каждому GIF

- Длительность: 8-18 секунд.
- Формат: `16:9`, желательно `1280x720`.
- Скорость: без ускорения x2/x3, чтобы шаги читались.
- Курсор: видимый, движения плавные, без резких рывков.
- В кадре: только релевантные действия, без лишних вкладок и уведомлений.
